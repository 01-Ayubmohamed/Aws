# Highly availaible Application Load balaner web app

---
## Objective 
This project will deploy an ALB behind two instances. Incorporating several AWS techniques demonstarting load balancing, health checks and a thorough secruity group isolation. The criteria that must be met for this project; include VPC with 4 subnets 2 public/private, 1 bastion host and two private EC2 instances. 1 ALB that controls all resource traffic to internet, security groups that ensure private resources grant access only to ALB, while interenet is only accessble through the ALB. An Auto Scalling Group (ASG) attached to private EC2 instance.
